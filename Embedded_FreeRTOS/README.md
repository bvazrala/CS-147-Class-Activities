# C04 — Embedded Characteristics and FreeRTOS Tasks

## Goal

Create two concurrent tasks: one samples an input and one reports status.

## Student Task

Open the `.ino` file and complete each `TODO` block. The intended coding load is small enough to complete during lecture.

## Expected Output

The ESP32 should print useful status information to the Serial Monitor. Activities with the OLED should also show the status on the screen.
